1. Window : 
    a. Window is the main JavaScript object root, aka the global object in a browser.
    b. It can be treated as the root of the document object model.
    c. Window is the first thing that gets loaded into the browser. 
    d. This window object has the majority of the properties like length, innerWidth, innerHeight, name,
       if it has been closed, its parents, and more.
    e. Since window is the global object you can reference any properties of it with just the property name,
       so you do not have to write down window it will be figured out by the runtime.

2. Screen : 
    a. window.screen or just screen is a small information object about physical screen dimensions.
    b. The portion of a screen displaying the rendered document is the viewport in JavaScript.
    c. The window object also has a screen object with properties describing the physical display like height and width.

3. Document : 
    a. window.document or just document is the main object of the potentially visible document object model(DOM).
    b. The document object is your html, aspx, php, or other document that will be loaded into the browser.
